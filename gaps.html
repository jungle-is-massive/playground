Revenue value on quote jobs
82 jobs affected

The Revenue field (labelled "Revenue" in Synergist, field code jobCustomField6) is empty on every single Quote job. This means the pipeline panel currently shows no £ values — the total pipeline value reads as zero. This is the single most important gap to fix.

Term type (Retainer vs Campaign)
55 jobs affected

The Term Type field distinguishes retainer work from campaign/project work. 46 quote jobs and 9 live jobs have this blank. Without it, the filters in both dashboard tabs can't split work by type — everything shows as "Untagged".

Production & diary data
Tab not yet buildable

The Production tab requires diary/calendar data from Synergist — specifically, resource bookings that show who is allocated to which shoot, edit, or delivery on which day. The Synergist Diary API exists, but we need to confirm the data is actually being entered and what it looks like for Jungle's production team.

Questions to answer before building
1
Is the production team using Synergist's diary/calendar to log resource bookings? If yes, who? If no, where do they track shoots?
2
How are shoots tagged in Synergist — by job type, phase, charge code, or something else? We need a consistent way to filter "shoot days" vs "edit days" vs "delivery".
3
Which team members need to appear in the resource/calendar view? Is it just production, or does it include account handlers and new biz too?
